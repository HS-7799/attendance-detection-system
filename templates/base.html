What are the advantages of this platform ?

This application makes it possible to manage student access to exam rooms in ENSAK in a secure way, using an IOT platform designed by Prof. BAJIT.

  • 01 Portability

    This platform was designed in a Linux environment but can still run on Windows. To be able to do that, you will only need to satisfy a few dependencies.

  • 02 Robustness

    Robustness is one of the strong points in our platform. It was designed to run even in abnormal conditions.

  • 03 Low price

    This platform was created with the strict minimum of material needed, making its cost very low.

 

Admin Dashboard

This Dashboard gives a global view on the platform and allows the Administrator to perform multiple actions and alter the application settings.

The Admin can therefore set up multiple aspects and details of the platform using the Dashboard.

Thanks to its elaborate design, the Admin won't face any difficulties in managing the different functionalities of this application.

Functionalities

The platform contains an array of functionalities, including these main ones:

Presence Detection

Once a person is detected, the process is turned on.

Health Check

An innovative functionality that allows the containment of the pandemic.

Radio Frequency Identification

Only registered students with cards are authorized access to the exam room.

Facial Recognition

Identifying students using image processing.
